CruiseControl.NET : Farm Report Farm Plugin
This page last changed on Apr 01, 2009 by csut017.

The Farm Report Farm Plugin shows you status information for all projects on all servers. If the Dashboard cannot connect to any specific server then an errors table is shown detailing the problems.

Example

Minimalist example: 

<farmReportFarmPlugin />

Full example: 

<farmReportFarmPlugin defaultSort="ServerName" />

Configuration Elements

Node Description Type Required Default
defaultSort The default column to sort on.
This setting is only available from release 1.4.4 onwards. 		One of the following values:
  • Name
  • BuildStatus
  • LastBuildDate
  • ServerName
  • Category
 No Name
