CruiseControl.NET : FTp Source control
This page last changed on Jun 28, 2009 by williams.

The Ftp Soure control block allows to detect new and changed files at an Ftp site, deleted files are NOT detected.

Full configuration example:

<sourcecontrol type="ftpSourceControl">
  <serverName>ftp.isp.com</serverName>
  <userName>john</userName>
  <password>doe</password>
  <ftpFolderName>config</ftpFolderName>
  <localFolderName>d:\temp\config</localFolderName>
  <recursiveCopy>true</recursiveCopy>
</sourcecontrol>

serverName the name of the server to connect to string true
N/A
