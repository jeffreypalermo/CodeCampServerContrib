CruiseServer Control Task

Sends a management task to a CruiseServer instance.

Version

This task is available from version 1.5.0.

Examples

Minimalist example:

<cruiseServerControl>
  <actions>
    <controlAction type="StartProject" project="CCNet" />
  </actions>
</cruiseServerControl>

Full example:

<cruiseServerControl>
  <actions>
    <controlAction type="StartProject" project="CCNet" />
  </actions>
  <description>Example of how to send a CruiseServer command.</description>
</cruiseServerControl>

Configuration Elements

Node Description Type Required Default Version
actions The actions to send. ControlAction (see below) Yes n/a 1.5.0
description A description of this task. String No None 1.5.0
dynamicValues Any dynamic parameters to be used. See Dynamic Parameters Dynamic values array - see Dynamic Parameters No None 1.5.0

Control Action Elements

There must be one or more control actions to send to the server. The will consist of the project name and the action to perform.

Node Description Type Required Default Version
type The type of action to send. One of the following values:
  • StartProject
  • StopProject
 Yes n/a 1.5.0
project The project the command is for. String Yes n/a 1.5.0
