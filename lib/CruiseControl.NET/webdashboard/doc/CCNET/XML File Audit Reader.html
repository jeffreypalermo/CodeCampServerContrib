XML File Audit Reader

Reads audit logging information from a file. The information must be stored in an XML format.

This reader handles reading audit information that has been written by the XML File Audit logger.

Version

This security item is available from version 1.5.0.

Examples

<auditReader type="xmlFileAuditReader" location="c:\Logs\ccnet_audit.log"/>

Configuration Elements

Node Description Type Required Default Version
type The type of the audit reader - must be xmlFileAuditReader. String Yes n/a 1.5.0.
location The location of the file . String No SecurityAudit.xml in the application folder 1.5.0
