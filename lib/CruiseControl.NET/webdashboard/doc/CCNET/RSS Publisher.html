CruiseControl.NET : RSS Publisher
This page last changed on Jun 26, 2009 by csut017.

This publisher generates an RSS file reporting the latest results for a Project.
The RSS feed is available via the Dasboard in the Project Report. There needs to be 1 build done with this publisher for the icon to show up.

Version

This publisher is available from version 1.3.0.

Minimal Configuration

<publishers>
   <rss/>
</publishers>

Full Configuration

<publishers>
   <rss items="30"/>
</publishers>

Configuration Elements:

Node Description Type Required Default Version
Items The number of items to allow in the feed. int false 20 1.4.4
dynamicValues Any dynamic parameters to be used. See Dynamic Parameters Dynamic values array - see Dynamic Parameters No None 1.5.0
